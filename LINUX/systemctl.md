##### ```systemctl``` 명령어

```systemctl status [패턴...|PID...]```
```systemctl start [유닛...]```
```systemctl stop [유닛...]```
```systemctl reload [유닛...]```
```systemctl restart [유닛...]```
```systemctl enable [유닛...]```
```systemctl disable [유닛...]```
```systemctl [유닛...|경로...]```
```--no-pager``` 페이지뷰 없이 출력
