`show ip router`: 라우팅 테이블을 확인
`show ip protocols`: 라우팅 프로토콜 확인
`show ip route [IP 주소]`: 해당 주소까지의 라우팅 경로
#### Static
```
Router(config)#ip route 0.0.0.0 0.0.0.0 Serial 0/0/0
```
`ip route [네트워크IP][서브넷마스크][인터페이스]`
#### RIP
```
Router(config)#router rip
Router(config-router)#version 2
Router(config-router)#network 192.168.2.0
```
version2부터 서브네팅을 지원한다.
`network [네트워크IP]`
#### OSPF
```
Router(config)#router ospf 10
Router(config-router)#network 192.168.1.0 0.0.0.255 area 0

Router#sh ip ospf interface
```
라우팅 프로토콜 설정에 진입할 때 ospf는 프로세스 ID를 정해 주어야 한다.
`network [네트워크IP] [와일드카드마스크] area [area번호]`
