##### 접속하기
```mysql -u root -p```

##### 버전확인
```SELECT VERSION();```

```
+-----------+
| VERSION() |
+-----------+
| 8.4.8     |
+-----------+
1 row in set (0.00 sec)
```

```

| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
+--------------------+

```
##### 데이터베이스에 들어가기
```use [db이름];```
##### 선택한 데이터베이스 속 테이블 목록 보기
```show tables;```
##### 테이블 내용 전체 보기 
```show * from [테이블이름];```
##### 테이블 내 특정 필드들만 보기 
```show [필드이름1],[필드이름2],... from [테이블이름];```
##### 데이터베이스 추가
```create database [db이름]```
##### 데이터베이스에 테이블 추가
```create table [테이블이름] ([칼럼이름1] [변수타입1], [필드이름2] [변수타입2],...)```
##### 테이블에 내용 추가
```insert into [테이블이름] values ([필드1값], [필드2값], ...)```
